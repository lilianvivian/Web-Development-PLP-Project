Our Work

A selection of projects that showcase our skills, passion, and commitment to quality.

Industrial IoT Turbine Monitor

A leading energy provider needed a way to proactively monitor wind turbine performance to prevent costly, unexpected failures and optimize maintenance schedules.

LiSkyBot developed a real-time, IoT-based monitoring system that tracks turbine speed, temperature, and vibrations. The system uses predictive analytics to flag potential issues and sends automated alerts to maintenance crews.

IoT React Node.js Real-Time Data Alert System

AI-Powered Fair Recruitment Platform

A major corporation was struggling with unconscious bias in their hiring process and spending thousands of hours manually screening resumes.

We built an AI-powered platform that analyzes resumes for skills and experience while masking identifying information (like name and gender) to mitigate bias. The system ranks candidates based on job requirements, creating a fair and efficient shortlist.

Artificial Intelligence NLP Python React Bias Detection

Dynamic Music Discovery App

A music startup wanted to create a more engaging way for users to discover and share music beyond standard, static playlisting.

We developed an interactive web application using the Spotify API that allows users to create dynamic playlists based on mood, genre, and activity. It features a clean user interface and seamless social sharing capabilities.
